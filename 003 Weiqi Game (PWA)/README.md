# 🎮 Simple Go Game Project

A beginner-friendly Go (Weiqi) game you can play in your browser!

## 📁 Files in This Project

1. **index.html** - The game board and buttons
2. **styles.css** - How the game looks and feels
3. **game.js** - How the game works and makes decisions
4. **PRD.md** - What we're building and why
5. **README.md** - This file (instructions)

## 🚀 How to Play

### Step 1: Open the Game
1. Double-click **index.html** to open it in your web browser
2. (OR open a folder, right-click index.html, and choose "Open with your browser")

### Step 2: Start Playing
1. Click **"🚀 Start Game"** button
2. Click any empty square on the board
3. Your black stone will appear!

### Step 3: Play Against Computer
1. Computer will think for a moment
2. Computer places a white stone
3. Continue taking turns until someone wins!

### Step 4: Win!
- Try to fill more squares than the computer
- Computer will automatically remove stones that can't escape
- Count your stones at the end to see who won!

## 🎯 Features

- **9x9 Board** - Small enough to be quick and fun
- **Computer Opponent** - Computer plays white stones
- **Score Tracking** - See who has more stones
- **Game Over Detection** - Game ends when board is full
- **Give Up Option** - Quit anytime if you want
- **New Game** - Start over anytime

## 🎨 What You'll Learn

This game teaches you:
- How to create HTML (structure)
- How to style it with CSS (appearance)
- How to make it work with JavaScript (logic)
- How to create a simple game
- How a computer can play against you

## 💡 Tips

- Try to place stones to surround squares
- Sometimes losing one stone can lead to a capture!
- Fill empty squares to increase your score

Have fun playing! 🎉
